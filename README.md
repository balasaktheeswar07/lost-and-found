# Lost and Found Website

This repository contains a simple Lost & Found web application built with HTML, CSS, and JavaScript.

## Project Details
- Fully responsive single-page website.
- Local storage persistence for reported items.
- Search/filter controls for Lost and Found items.
- Accessible modal forms and improved UI/UX.
- Removed background music (BGM) feature for a cleaner browsing experience.

## Files
- `OneDrive/Documents/index.html` — main landing page.
- `OneDrive/Documents/style.css` — site styling and responsive layout.
- `OneDrive/Documents/script.js` — form handling, item rendering, search/filter, and chatbot interaction.

## Notes
The background music feature has been removed from the application and is no longer documented in this README.
